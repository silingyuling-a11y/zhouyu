**常见原因及解决方法：**

1. **光猫问题**（最常见）
   - 看看光猫上的"LOS"灯是不是红的，如果红灯闪就是光缆断了
   - 解决：拔掉光猫电源，等30秒再插上，重启一下试试
   - 如果重启后还是红灯，那就是线路问题，需要报修

2. **路由器问题**
   - 光猫正常但WiFi不能用，大概率是路由器的问题
   - 解决：重启路由器，检查一下网线有没有插好
   - 如果还是不行，可能路由器坏了，需要换

3. **欠费停机**
   - 突然断网，而且之前没出过问题，先查查是不是欠费了
   - 解决：交了话费一般10分钟内就会恢复

4. **大面积故障**
   - 如果整个小区都断了，那可能是光缆被挖断了或者机房故障
   - 解决：打10000号报障，或者找你的装维师傅

**简单排查步骤：**
1. 看光猫灯（LOS红灯=线路问题，PON灯不亮=认证问题）
2. 重启光猫和路由器
3. 检查有没有欠费
4. 直接用网线连电脑试试（排除WiFi问题）

3. **光猫问题**
   - 光猫太老，不是千兆光猫
   - 光猫没放在通风的地方，过热降速
   - 解决：重启光猫，或者联系师傅换千兆光猫

4. **运营商侧问题**
   - 这个概率其实很低，电信的网络还是比较稳的
