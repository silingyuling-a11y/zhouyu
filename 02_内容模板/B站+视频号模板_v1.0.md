```
- 总结核心观点
- 引导三连（点赞投币收藏）

```

## 视频号模板

### 模板4：短平快科普型（15-60秒）

#### 视频标题
- 在广州装宽带，记住这3点不踩坑
- 广州宽带最便宜的办理方式

- 网速慢？教你一招快速解决
- 广州宽带办理攻略，建议收藏

#### 视频结构
```
【0-3秒：钩子】
- 直接抛问题
- 或者痛点画面

【3-40秒：干货输出】
- 讲1-3个知识点
- 语速快、信息密度高
- 配字幕、配画面

【40-55秒：引导互动】
- 总结一下
- 关注我，了解更多宽带知识
- 有问题评论区留言
```

#### 字幕要求
- 大字体、醒目
- 重点内容标红或加粗
- 每屏信息不要太多
- 配合emoji增加趣味性

## 视频平台运营技巧

### B站运营
1. **标题优化**：关键词前置+吸引眼球的词
2. **封面设计**：大字标题+对比图/表情包
3. **标签设置**：核心关键词+相关标签，10个左右
4. **分区选择**：科技区、生活区、知识区
5. **互动引导**：开头结尾引导三连、评论
6. **系列内容**：做系列视频，增加关注率
7. **专栏配合**：视频同步发图文专栏，双份流量

### 视频号运营
1. **标题党**：疑问句、数字、痛点
2. **封面**：人脸+大字，朋友圈里一眼看到
3. **话题标签**：#广州宽带 #电信宽带 #宽带办理
4. **地理位置**：定位广州，本地流量更多
5. **引导分享**：结尾说"转发给需要的朋友"
6. **公众号联动**：视频号挂公众号链接
7. **私域导流**：引导加微信/进粉丝群

### 通用技巧
- **内容质量**：清晰的画面、清楚的声音、有价值的内容
- **发布时间**：晚上7-10点发布，看的人最多
- **持续更新**：定期更新，保持账号活跃度
- **评论互动**：及时回复评论，提高互动率
- **数据分析**：看播放量、完播率、点赞率，优化内容
